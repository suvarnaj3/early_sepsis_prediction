# early_sepsis_prediction
# Sepsis Prediction using Machine Learning

## 📌 Overview
This project predicts early onset of sepsis using ICU patient data.

## ⚙️ Techniques Used
- Feature Engineering (time-series to patient-level)
- SMOTE for class imbalance
- Class weighting

## 🤖 Models Used
- Logistic Regression
- Random Forest
- XGBoost
- LightGBM

## 📊 Results
- Best Model: XGBoost / LightGBM
- AUROC: ~0.88
- PR-AUC: ~0.66

## 📁 Dataset
PhysioNet Sepsis Dataset (not included due to size)

## 🚀 How to Run
1. Open notebook in Google Colab
2. Mount Google Drive
3. Run all cells
